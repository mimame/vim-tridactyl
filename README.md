# vim-tridactyl [![Vint][vintbadge]][vintstatus]

VIM syntax plugin for [Tridactyl] configuration files.

## Installation

Install with your favorite VIM plug manager. E.g. for [vim-plug][]:

``` vim
Plug 'tridactyl/vim-tridactyl'
```
